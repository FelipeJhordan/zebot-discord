# zebot-discord
## Descrição
Boot criado com node com objetivo de utilizar em um canal de discord.


## Como executar um comando
Todo comando deve iniciar com o prefixo "-" e após isso o comando.                                                 
Exemplo: "-send-hello"

## Lista de comandos atuais
- -send-hello
- - -fish

## Requisitos
- Node version 16.13.2
